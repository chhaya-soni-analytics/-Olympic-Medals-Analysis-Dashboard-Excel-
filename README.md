# 🏅 Olympic Medals Analysis Dashboard (Excel)

## 🚀 Project Overview

This project focuses on analyzing historical Olympic medal data to uncover patterns in **country performance, athlete achievements, gender participation, and sport-wise dominance**.

An interactive Excel dashboard was developed to transform raw Olympic data into meaningful and visually engaging insights.

---

## 🎯 Objectives

* Analyze **total medal distribution (Gold, Silver, Bronze)**
* Identify **top-performing countries and athletes**
* Study **gender-wise participation and performance**
* Explore **sport-wise medal contribution**
* Understand **historical trends in Olympic performance**

---

## 📊 Dashboard Features

### 🔹 Key Metrics (KPIs)

* Total Medals
* Gold, Silver, Bronze Medals
* Total Athletes
* Participation Overview

---

### 📈 Visualizations

* Medal Distribution by Type (Donut Chart)
* Country-wise Medal Count (Bar Chart)
* Gender-wise Medal Comparison
* Top Athletes by Medal Count
* Sport-wise Medal Analysis
* Medal Distribution Map (Geographical View)

---
